# Excalidraw GitHubエディター - Vibe Coding実験

[English](./README.md) | [简体中文](./README.zh-CN.md)

このプロジェクトは、ユーザーがGitHubリポジトリを閲覧し、Excalidrawエディターを使って `.excalidraw` ファイルをブラウザ内で直接開き、編集して保存できる実験的なWebアプリケーションです。

**このプロジェクトの主な意義は、機能だけでなく、その制作プロセスにもあります：本プロジェクトは全てVibe Coding（AIコーディングアシスタントを活用した対話型開発）によって作られました。**

## 主な機能

* **GitHubリポジトリの閲覧**: `repo` スコープ付きのパーソナルアクセストークン(PAT)でGitHubアカウントに接続。
* **リポジトリ＆ブランチ選択**: アカウント内のリポジトリとブランチを選択可能。
* **ファイルツリーナビゲーション**: 選択したブランチのファイル構造を表示。
* **Excalidrawファイル編集**: ファイルツリーで `.excalidraw` ファイルをクリックすると、埋め込みExcalidrawエディターで開く。
* **ブラウザ内キャッシュ**: IndexedDBで現在の描画状態をキャッシュし、ファイル切り替え時もローカルの未保存変更を保持。
* **変更検知**: 内容比較により変更されたファイルはツリー上にアスタリスク（`*`）で表示。
* **GitHubへ保存**: 変更をカスタムコミットメッセージ付きでGitHubリポジトリに保存。
* **ファイル操作**: 新規ファイル作成、既存ファイルのリネーム/削除（フォルダー操作はガイド付き）。
* **多言語対応(i18n)**: 英語・中国語インターフェース対応。ブラウザの言語設定を自動検出。
* **（計画中）ファイルのコミット履歴閲覧**

## はじめ方

1. **リポジトリをクローン**
    ```bash
    git clone https://github.com/icattlecoder/excalidraw-gh.git
    cd excalidraw-gh
    ```
2. **依存関係をインストール**
    ```bash
    npm install
    ```
3. **GitHub Personal Access Token(PAT)を生成**
    * [GitHubのDeveloper settings](https://github.com/settings/tokens)へ
    * 新しい **classic** トークンを生成
    * 名前を付ける（例：Excalidraw Editor）
    * `repo` スコープを選択
    * 生成されたトークンをコピー（再表示不可）
4. **開発サーバーを起動**
    ```bash
    npm run dev
    ```
5. ブラウザで指定されたローカルURL（通常`http://localhost:5173`）を開く
6. PATを入力（トークンはIndexedDBにローカル保存され、GitHub API以外に送信されません）
